# 大OS 報價系統 BIBLE（公開去敏感化版）

聚陽「大OS」（Quotation v3）操作說明的**對外候選版**。本版已移除客戶／成本／個資，供評估是否可公開使用。

## 這一版做了什麼去敏感化

- **只保留**確認無客戶/成本/個資的畫面：封面、安裝、三階層結構圖、報價流程圖、工具列、選單（6 張）。
- **移除**所有含客戶名／成本／供應商／數量／同事姓名的操作截圖（Notion、Group Level 1/3、Dev.Style、Option Level 1/2、複製、TP Compare、舊報價轉入）——改以文字說明＋「公開版已省略」提示卡呈現。
- **Merge Options** 章不用真實截圖，改用**自製示意圖**（A/B/C 通用數量）。
- 內文移除點名客戶（GAP Inc → 特定客戶）與真實數量。

## ⚠️ 仍需確認才可對外

- 本版為 **AI 去敏感化的候選版**，**對外公開前仍需相關部門核准**（屬「需相關部門確認」事項）。
- 提醒：GitHub Pages 在免費／個人帳號上發布的網站**一律公開**（私有 Pages 需 Enterprise 方案）。公開即任何人可存取並被搜尋引擎收錄。
- 若僅供公司內部使用，建議直接用**內部完整版**（含真實截圖）放內部平台，不需走公開。

## 本機預覽

```bash
python -m http.server 8080   # 在本資料夾執行，瀏覽器開 http://localhost:8080
```

## 部署（取得核准後）

於 **Settings → Pages** 將 Source 設為 `main / (root)`。網址形如 `https://<帳號>.github.io/<repo>/`（記得帶 repo 名與結尾斜線）。
